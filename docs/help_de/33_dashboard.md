# Dashboards

Die **Dashboards** Seite zeigt Portfolio-Performance-Visualisierungen für deine Live-Passivbot-Instanzen.
Du kannst mehrere Dashboards erstellen und über die Sidebar zwischen ihnen wechseln.

---

## Dashboard anzeigen

- Existiert nur ein Dashboard, wird es automatisch geöffnet.
- Bei mehreren Dashboards erscheint im Hauptbereich eine **Auswahlbox** — das gewünschte Dashboard auswählen.
- Alle Dashboards sind außerdem als Schnellzugriff-Schaltflächen in der Sidebar aufgelistet.

---

## Sidebar-Aktionen

| Schaltfläche | Aktion |
|--------|--------|
| `:material/refresh:` | Aktuelles Dashboard neu laden |
| `:material/add_box:` | Neues leeres Dashboard erstellen |
| `:material/edit:` | Aktuell angezeigtes Dashboard bearbeiten |

Im **Bearbeitungsmodus** wechselt die Sidebar zu:

| Schaltfläche | Aktion |
|--------|--------|
| `:material/save:` | Änderungen auf Disk speichern |
| `:material/cancel:` | Änderungen verwerfen und vorherigen Zustand wiederherstellen |
| `:material/delete:` | Aktuelles Dashboard dauerhaft löschen |

---

## Dashboard erstellen

1. `:material/add_box:` in der Sidebar klicken.
2. Im Bearbeitungsformular Widgets und Datenquellen konfigurieren.
3. `:material/save:` klicken — das Dashboard wird gespeichert und sofort angezeigt.

---

## Dashboard bearbeiten

1. Das gewünschte Dashboard öffnen (Name in der Sidebar klicken).
2. `:material/edit:` klicken — das Bearbeitungsformular öffnet sich.
3. Charts, Benutzer oder Layout anpassen.
4. `:material/save:` zum Übernehmen oder `:material/cancel:` zum Verwerfen klicken.

---

## Navigation von anderen Seiten

Andere PBGui-Seiten können direkt zum Dashboard navigieren, das am stärksten einem bestimmten Benutzerkonto zugeordnet ist.
Die Seite wählt automatisch das Dashboard aus, das diesen Benutzer am stärksten referenziert.
