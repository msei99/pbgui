# Dashboards

The **Dashboards** page shows portfolio performance visualisations for your live Passivbot instances.
You can create multiple dashboards and switch between them from the sidebar.

---

## Viewing a dashboard

- If only one dashboard exists it opens automatically.
- If multiple dashboards exist, a **selectbox** appears in the main area — pick the dashboard you want to view.
- All dashboards are also listed as quick-access buttons in the sidebar.

---

## Sidebar actions

| Button | Action |
|--------|--------|
| `:material/refresh:` | Reload the current dashboard |
| `:material/add_box:` | Create a new empty dashboard |
| `:material/edit:` | Edit the currently displayed dashboard |

When **editing** a dashboard the sidebar switches to:

| Button | Action |
|--------|--------|
| `:material/save:` | Save changes to disk |
| `:material/cancel:` | Discard changes and restore the previous state |
| `:material/delete:` | Delete the current dashboard permanently |

---

## Creating a dashboard

1. Click `:material/add_box:` in the sidebar.
2. In the edit view, configure the widgets and data sources.
3. Click `:material/save:` — the dashboard is saved and shown immediately.

---

## Editing a dashboard

1. Open the dashboard you want to change (click its name in the sidebar).
2. Click `:material/edit:` — the edit form opens.
3. Adjust charts, users, or layout.
4. Click `:material/save:` to apply changes, or `:material/cancel:` to revert.

---

## Navigation from other pages

Other PBGui pages can link directly to the dashboard most relevant to a specific user account.
The page will automatically select the dashboard that most strongly references that user.
