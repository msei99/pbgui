# v1.85

## Highlights

- DB Tools now include a FastAPI cleanup and migration page, a full Backup Manager, and one-way Sync Jobs for copying users across known masters with manual runs, scheduling, multi-target fan-out, row-count verification, and readable per-job logs.
- VPS Manager adds Change UFW for local and remote masters with rule-list editing, quick allow rules for SSH/PBGui/OpenVPN, stale-rule protection, and lockout safety checks.
- PBData price feeds are more robust with Binance bookTicker fallback prices, stable Binance ticker receive loops, better timeout handling, and smaller Bybit price websocket chunks.
- Selection UIs were standardized across VPS Manager, DB Tools, API Keys, Dashboard, Dashboard Templates, Optimize V7, and Market Data.

## Improvements

- Services now expose systemd autostart state and Enable/Disable controls also start or stop the service as expected.
- DB Tools can restart local PBData through its systemd user service during database operations.
- PBGui branch switching in VPS Manager now handles tracked local changes and fetches all origin branches before checkout.
- VPS Manager can discover remote PBGui branches on single-branch master installs without manual git remote configuration.
- VPS Monitor records local master CPU history from the monitor loop and closes metric stream SSH subprocesses more cleanly.
- The Services Price Snapshot overlay auto-refreshes and handles millisecond timestamps correctly.
- Dashboard Balance and Positions widgets load database data first and refresh live data in the background when the scope allows it.

## Fixes

- Restart clicks in VPS Manager and movable log viewers remain reliable while websockets reconnect.
- Background 401 responses no longer redirect the shared navigation until the session is confirmed invalid.
- VPS password prompts now show the target host and VPS user more clearly.
