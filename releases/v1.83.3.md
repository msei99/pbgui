# v1.83.3

## Highlights

- VPS Manager: fixed Setup VPS form editing so live VPS metadata refreshes no longer reinsert stale PBRemote bucket or CoinMarketCap API-key values after clearing optional fields.
- VPS Manager: fixed focus-triggered refreshes so clicking into the CoinMarketCap API-key field no longer restores a just-cleared PBRemote bucket.
- VPS Manager: kept `Save VPS` enabled while cleared optional PBRemote/CoinMarketCap fields differ from the saved VPS configuration.
- VPS Manager: `Save VPS` now automatically applies changed optional PBRemote/CoinMarketCap settings to the remote VPS `pbgui.ini` and starts or stops the optional services accordingly.
- VPS Manager: automatically opens the VPS task log when `Save VPS` starts the optional settings apply job.
- PBv7: disabled and backend-blocked `enabled_on` targets without confirmed CoinMarketCap support while `dynamic_ignore` is enabled.
