# v1.83.1

## Highlights

- VPS Manager: allow new VPS setup without a PBRemote bucket or CoinMarketCap API key and skip the optional PBRemote/PBCoinData service setup when those values are empty.
- VPS Monitor: treat unconfigured optional PBRemote/PBCoinData services as skipped, prevent restarts without required config, and block dynamic-ignore V7 sync/startup on hosts without a CoinMarketCap API key.
- VPS Manager: auto-fill the Add VPS IPv4 field from local `/etc/hosts` when the entered hostname already resolves there.
- VPS Manager: show the Add VPS hostname field before IPv4 so `/etc/hosts` auto-fill can be used first.
- VPS Manager: prefill Add VPS Allowed SSH IPs with the master's detected public IPv4 address.
