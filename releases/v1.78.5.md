# v1.78.5

Released: 20-05-2026

- Reorganized the v7 web editor live settings into smaller Advanced subsections and added dedicated GUI fields for newer PB7 runtime parameters such as fill overlap, warmup, market snapshot strategy, candle freshness, and forager stability.
- Updated the v7 web editor fallbacks to current PB7 schema defaults and now prevent invalid `max_n_cancellations_per_batch` / `max_n_creations_per_batch` combinations during editing instead of only rejecting them on save.
- Clarified the v7 web editor help tooltips for the batch cancel/create fields so the automatic range limiting is visible to users.
- Fixed remaining v7 web editor PB7-schema mismatches in field help/input constraints, including the `recv_window_ms` default tooltip and allowing `max_ohlcv_fetches_per_minute = 0` as documented.
- Bumped the API serial so already running processes clearly show that a restart is needed before FastAPI-injected version strings and related release metadata fully reflect `v1.78.5`.
- Clarified the repository agent release workflow so future releases always include the `api/serial.txt` bump as a standard release-preparation step.
