# v1.83.2

## Highlights

- VPS Manager: fixed PBGui updates so clearing the PBRemote bucket or CoinMarketCap API key is written to the remote `pbgui.ini` before optional services are restarted.
- VPS Manager: fixed settings refresh to read the PBRemote bucket from the VPS `pbgui.ini` instead of copying the master's bucket into the VPS configuration.
- VPS Manager: synced VPS PBRemote bucket and CoinMarketCap API key from live VPS metadata so secondary masters keep their local VPS configuration current.
- VPS Manager: blocked removing a VPS CoinMarketCap API key while running v7 `dynamic_ignore` bots on that VPS still depend on it.
