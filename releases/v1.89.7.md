# v1.89.7

- Fixed queued PB7 backtests so `coin_overrides` files such as `HYPE.json` are copied beside the queued config snapshot before launch.
- Added a VPS Manager purge action that removes PBGui/PB7 install files and PBGui-managed remote autostart artifacts before deleting a VPS record locally.
- Fixed Cluster Sync materialization so newer node updates, such as disabling a VPS before removal, are not overwritten by older peer operations.
- Fixed Cluster Sync node removal so later peer membership updates cannot recreate a removed node as a partial `n/a` row.
